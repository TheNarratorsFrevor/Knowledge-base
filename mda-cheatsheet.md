## 1. Foundational Structures

### Groups (Additive)
- Nonempty $G$, $+: G\times G \to G$
- Assoc: $(x+y)+z=x+(y+z)$
- ID $0$: $0+x=x$
- Inv $-x$: $x+(-x)=0$
- Abelian: $x+y=y+x$
- **Ex**: $(\mathbb{Z},+), (\mathbb{R},+)$ ‚úì; $(\mathbb{N},+)$ ‚úó (no inv)
- **Subgroup** $H\leq G$: $0\in H$, $-x\in H$, closed $x+y\in H$
- Proper $H\subsetneq G$; triv $\{0\}, G$
- $H\cap K$ subgroup; $H\cup K$ subgroup $\iff H\subseteq K$ or $K\subseteq H$
- Ex: $2\mathbb{Z} \leq \mathbb{Z}$ [file:1]

### Groups (Multiplicative)
- Same w/ $\cdot$, ID $1$, inv $x^{-1}$
- Ex: $(\mathbb{R}\setminus\{0\}, \cdot)$ [file:1]

### Fields $K$
- $(K,+)$ abelian w/ $0$; $(K\setminus\{0\}, \cdot)$ abelian w/ $1$
- Distrib: $(x+y)z = xz + yz$
- **Ex**: $\mathbb{R}, \mathbb{Q}$ ‚úì; $\mathbb{Z}$ ‚úó (no mult inv)
- **Subfield**: inherits all ops [file:1]

## 5. Linear Operators & Matrices

### Linear Map $f:U\to V$
$$f(\lambda u_1 + \mu u_2) = \lambda f(u_1) + \mu f(u_2)$$
- **Kernel**: ker$f$={u|f(u)=0} ‚â§ $U$
- **Image**: im$f$=span{f(u)} ‚â§ $V$
- Injective iff ker$f$={0}; surjective iff im$f$=$V$
- **Rank-Nullity**: dim$U$ = dim(ker$f$) + dim(im$f$) [file:1]
